PENGEMBANGAN BOUNCHING BALL GAME 
Pendahuluan 
Bouncing ball merujuk pada suatu permainan yang mengarahkan bola ke target terdekat dan hal ini telah di kembangkan beberapa tahun yang lalu sehingga banyak sekali pengembangan game ini memilih membuat game ini karena perancangan yang lebih simple dan mudah dimainkan. Adapun pula pengembangan game yang dirancang untuk melatih kemampuan dalam menggunakan C++ sebagai bentuk program dengan mengutamakan titik koordinat tertentu agar game dapat dijalankan sesuai denga napa yang kita perintahkan. Tujuan dari pengembangan ini ialah melatih kemampuan dalam menganalisis titik koordinat menggunakan pemrograman C++, merancang suatu konsep, pengembangan desain, grafik serta interaktif dari pengembangan Tingkat lanjut. 

Fitur-fitur Dalam Pengembangan Game 
Pada Bouncing Ball, terdapat fitur-fitur yang memungkinkan game berjalan dengan lancer meskipun ini baru memasuki perencanaan awal namun dipastikan pada pengembangan game tersebut dapat membuahkan hasil yang sesuai berdasarkan apa yang direncangakan antara lain: 
•	Start : sebagai variable awal sebelum memulai atau meluncurkan game yang akan kita jalankan. 
•	Setting : mengatur konfigurasi game yang akan kita ubah atau kita atur. 
•	Controller : fitur untuk mengendalikan arah bola yang kita kendalikan sesuai titik koordinat. 
•	End Game : sebagai akhir dari game yang telah kita jalankan. 

Penjelasan 
Pengembangan game awal ini dikembangkan sebagai pengembangan lanjut yang menggunakan pemrograman C++ dengan struktur class sebagai algoritma pemrograman sehingga dipilihlah untuk mengembangkan game bouncing ball tersebut. Dalam proses pengembangan ini pun dirancang berdasarkan variable-variabel angka untuk menentukan titik-titik koordinat pada kode program yang akan digunakan sehingga dapat menentukan objek pada algoritma tersebut. Pada algoritma objek dapat kita tentukan kemana arah titik koordinat yang diluncurkan namun harus berdasarkan stuktur variable pada algoritma program yang telah dirancang sebelumnya. 

Kesimpulan 
Game Bouncing Ball ini menunjukkan bagaimana konsep dasar grafis dan pergerakan objek dapat diimplementasikan dengan C++. Proposal ini menjadi dasar untuk pengembangan fitur lanjutan seperti kontrol pemain, skor, atau level kesulitan sehingga pemain dapat mengerti dalam mengontrol Gerak game. 

Penutup 
Proposal ini diharapkan dapat menjadi referensi awal bagi mahasiswa dalam mengembangkan game berbasis C++. Dengan fondasi sederhana, game dapat dikembangkan menjadi lebih kompleks dan menarik juga mengesankan ide-ide kreatif dalam pengembangan game lanjut.
